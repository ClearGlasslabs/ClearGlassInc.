# Secure Memory Architecture

## Objectives

- Tenant isolation
- Encryption at rest
- Scoped retrieval
- Regulated retention
- Auditability

## Memory Layers

| Layer | Technology |
|---|---|
| Session Memory | Redis |
| Workflow State | Cosmos DB |
| Vector Retrieval | Azure AI Search |
| Audit Archive | Azure Data Lake |

## Governance Controls

- RBAC enforced retrieval
- Purview classification
- PII-aware embeddings
- Tenant-scoped vector indexes
- Time-bound retention windows
